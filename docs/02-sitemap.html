URL一覧（reservation-app）
URL HTTPメソッド 画面/機能 説明
/ GET ルート導線 `index.html` へリダイレクトする入口。
/index.html GET 公開トップ画面 サービス案内・API利用導線を表示する公開ページ。
/login.html GET 管理ログイン画面 管理者の認証入力画面（エラー/ログアウト表示含む）。
/admin.html GET 管理画面（Backoffice） 予約・メルマガ情報の確認、集計表示、再読込、ログアウトを行う。
/login POST ログイン処理 Spring Securityの認証処理を実行し、成功時は管理画面へ遷移。
/logout POST ログアウト処理 セッションを破棄してログイン画面へ遷移。
/api/reservations/availability GET 空き枠照会API コース・期間条件で予約可能枠を取得する。
/api/reservations POST 予約登録API 予約情報を登録し、予約番号を返却する。
/api/mail/subscriptions/register POST メルマガ登録API メールアドレスをメルマガ配信対象として登録する。
/api/mail/subscriptions/unsubscribe POST メルマガ配信停止API メールアドレスの配信停止を登録する。
/api/admin/reservations GET 管理:予約一覧API 管理画面向けに直近予約一覧を取得する。
/api/admin/reservation-summary GET 管理:予約集計API 週次予約状況・月次推移などの集計情報を取得する。
/api/admin/mail-subscriptions GET 管理:メルマガ一覧API メルマガ登録/停止の一覧情報を取得する。
/api/admin/mail-summary GET 管理:メルマガ集計API 月次の登録数・停止数推移を取得する。
/api/admin/me GET 管理:ログインユーザーAPI 現在ログイン中の管理ユーザー情報を取得する。
