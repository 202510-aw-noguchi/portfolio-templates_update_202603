2 テーブル定義

テーブル名：reservation
カラム名 データ型 NULL デフォルト 制約 説明
id BIGINT NO AUTO_INCREMENT PK 予約ID
reservation_number VARCHAR(32) NO なし UNIQUE 予約番号（例: `RSV-YYYYMMDD-00001`）
course VARCHAR(20) NO なし なし コース種別（`classic` / `luxe`）
reserve_date DATE NO なし なし 予約日
reserve_time VARCHAR(20) NO なし なし 予約時間帯（例: `14:00-15:00`）
participant_count INTEGER NO なし なし 参加人数
contact_name VARCHAR(120) NO なし なし 申込者名
contact_phone VARCHAR(40) NO なし なし 連絡先電話番号
contact_email VARCHAR(160) NO なし なし 連絡先メールアドレス
created_at TIMESTAMP NO アプリ側で現在日時設定 なし 作成日時
テーブル名：slot_capacity
カラム名 データ型 NULL デフォルト 制約 説明
id BIGINT NO AUTO_INCREMENT PK 定員設定ID
reserve_date DATE NO なし なし 対象日
reserve_time VARCHAR(20) NO なし なし 対象時間帯
capacity INTEGER NO なし なし 当該枠の定員
テーブル名：mail_subscription
カラム名 データ型 NULL デフォルト 制約 説明
id BIGINT NO AUTO_INCREMENT PK 購読ID
email VARCHAR(160) NO なし UNIQUE 購読メールアドレス
active BOOLEAN NO なし なし 購読状態（`true`:購読中, `false`:解除済み）
created_at TIMESTAMP NO アプリ側で現在日時設定 なし 作成日時
updated_at TIMESTAMP NO アプリ側で現在日時設定 なし 更新日時
unsubscribed_at TIMESTAMP YES なし なし 解除日時（解除時のみ設定）

3 インデックス一覧

テーブル名 インデックス名 対象カラム 種類 設定理由
reservation PK_reservation id PRIMARY 主キー検索のため
reservation UQ_reservation_number reservation_number UNIQUE 予約番号の一意性保証・番号検索高速化
reservation IDX_reservation_course_date_time course, reserve_date, reserve_time NON-UNIQUE 空き枠計算（`sumParticipants`）の条件検索を高速化
reservation IDX_reservation_reserve_date reserve_date NON-UNIQUE 期間別予約取得（`findByReserveDateBetween`）を高速化
reservation IDX_reservation_created_at created_at NON-UNIQUE 作成日時範囲検索・新着順取得を高速化
slot_capacity PK_slot_capacity id PRIMARY 主キー検索のため
slot_capacity UQ_slot_capacity_date_time reserve_date, reserve_time UNIQUE 日付×時間帯の定員を一意に管理し、枠検索を高速化
mail_subscription PK_mail_subscription id PRIMARY 主キー検索のため
mail_subscription UQ_mail_subscription_email email UNIQUE メールアドレスの重複防止・登録/解除時の検索高速化
mail_subscription IDX_mail_subscription_updated_at updated_at NON-UNIQUE 管理画面の更新日時降順一覧取得を高速化
mail_subscription IDX_mail_subscription_created_at created_at NON-UNIQUE 月次登録推移（期間検索）を高速化
mail_subscription IDX_mail_subscription_unsubscribed_at unsubscribed_at NON-UNIQUE 月次解除推移（期間検索）を高速化
