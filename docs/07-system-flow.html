データの流れ
Browser -> Controller(API) -> Service -> Repository -> DB
Browser <- JSON Response <- Controller <- Service <- Repository <- DB

画面遷移の流れ
Browser -> /login.html, /admin.html (static)
admin.html -> /api/admin/* を fetch
registration.html -> /api/reservations/* を fetch
mail.html -> /api/mail/subscriptions/* を fetch

レイヤー構成の説明
各レイヤーの役割を表形式で説明
レイヤー 役割 主なクラス
クライアント（ブラウザ） 画面表示、ユーザー入力、API呼び出し（fetch）を担当 workshop/registration.html, mail.html, reservation-app/src/main/resources/static/admin.html, reservation-app/src/main/resources/static/login.html
Controller層 HTTPリクエスト受付、入力受け取り、Service呼び出し、レスポンス返却 ReservationController, MailSubscriptionController, AdminController, CsrfTokenController, RootRedirectController
Service層 業務ロジック（予約可否判定、集計、メルマガ登録/解除など）を実装 ReservationService, MailSubscriptionService
Repository層 DBアクセスを抽象化し、エンティティの永続化・検索を担当 ReservationRepository, MailSubscriptionRepository, SlotCapacityRepository
データベース層 永続データの保存・参照 H2（reservationdb）、エンティティ: Reservation, MailSubscription, SlotCapacity
View層（Thymeleaf） サーバーサイドHTMLテンプレート生成 なし（本アプリはThymeleaf未使用、静的HTML配信）

2 リクエスト処理フロー

管理画面「予約一覧」の処理フロー
[Client（ブラウザ）]
   |
   | 1) GET /api/admin/reservations
   v
[AdminController.reservations()]
   |
   | 2) 一覧取得を依頼
   v
[ReservationService.getRecentReservations()]
   |
   | 3) データ取得を依頼
   v
[ReservationRepository.findTop200ByOrderByCreatedAtDesc()]
   |
   | 4) SELECT ... FROM reservation ...
   v
[Database（H2）]
   |
   | 5) 予約一覧データ返却
   v
[ReservationRepository]
   |
   | 6) Entity -> DTOへ変換して返却
   v
[ReservationService]
   |
   | 7) 一覧データをControllerへ返却
   v
[AdminController]
   |
   | 8) JSONレスポンス返却（200 OK）
   v
[Client（ブラウザ）]
   |
   | 9) admin.html が受信データを一覧描画
登録処理のフロー
[Client（入力画面: registration.html / form）]
   |
   | POST /reservations/register
   | フォーム送信
   v
[ReservationController.create(
    @Valid ReservationForm form,
    BindingResult bindingResult,
    Model model
)]
   |
   | 1) Bean Validation 実行（@NotBlank, @Email, @Min, @Max ...）
   |
   +-- bindingResult.hasErrors() == true --------------------------+
   |                                                               |
   | 2-E) エラー情報をModelへ設定                                     |
   |      model.addAttribute("form", form)                         |
   |      model.addAttribute("errors", bindingResult)              |
   |                                                               |
   | 3-E) 入力画面を再表示（戻る）                                     |
   |      return "reservation/form";                               |
   |                                                               |
   +---------------------------------------------------------------+
   |
   +-- bindingResult.hasErrors() == false ----------------------+
                                                                |
                                                                | 2-N) Serviceへ登録依頼
                                                                v
                                                    [ReservationService.create(form)]
                                                                |
                                                                | 3-N) Repositoryで保存
                                                                v
                                                    [ReservationRepository.save(entity)]
                                                                |
                                                                | 4-N) INSERT
                                                                v
                                                    [Database]
                                                                |
                                                                | 5-N) 保存完了
                                                                v
                                                    [ReservationController]
                                                                |
                                                                | 6-N) 完了画面/一覧へリダイレクト
                                                                |      return "redirect:/reservations/complete";
                                                                v
                                                    [Client（完了画面/一覧画面を表示）]
