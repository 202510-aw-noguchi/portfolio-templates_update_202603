1 パッケージ構成

src/main/java/
└─ com/example/reservation/
   ├─ api/
   │  └─ dto/
   ├─ config/
   ├─ model/
   ├─ repository/
   └─ service/
パッケージ 役割 使用するアノテーション
src/main/java Javaソース全体のルート なし
com.example.reservation アプリ起動・全体の基点 @SpringBootApplication
com.example.reservation.api REST APIのエンドポイント公開 @RestController, @RequestMapping, @GetMapping, @PostMapping, @RequestParam, @RequestBody, @Valid, @DateTimeFormat
com.example.reservation.api.dto API入出力DTO（リクエスト/レスポンス）定義 @NotBlank, @NotNull, @Min, @Max, @Pattern, @Email
com.example.reservation.config Web/Security/Beanなどの設定 @Configuration, @Bean, @Controller, @GetMapping, @Value, @Override
com.example.reservation.model JPAエンティティ（予約・枠・購読）定義 @Entity, @Table, @Id, @GeneratedValue, @Column, @PrePersist, @PreUpdate
com.example.reservation.repository DBアクセス（Spring Data JPA） @Query, @Param
com.example.reservation.service 業務ロジック・トランザクション制御 @Service, @Transactional

メソッド 引数 戻り値 SQL種別 説明
ReservationRepository.sumParticipants course:String, reserveDate:LocalDate, reserveTime:String int SELECT 条件一致する予約の participantCount 合計を取得（@Query）。
ReservationRepository.findTop200ByOrderByCreatedAtDesc なし List<Reservation> SELECT 予約を作成日時降順で最大200件取得。
ReservationRepository.findByReserveDateBetween from:LocalDate, to:LocalDate List<Reservation> SELECT 予約日が範囲内のレコードを取得。
ReservationRepository.findByCreatedAtBetween from:LocalDateTime, to:LocalDateTime List<Reservation> SELECT 作成日時が範囲内のレコードを取得。
SlotCapacityRepository.findByReserveDateAndReserveTime reserveDate:LocalDate, reserveTime:String Optional<SlotCapacity> SELECT 指定日付・時間枠の定員設定を1件取得。
MailSubscriptionRepository.findByEmail email:String Optional<MailSubscription> SELECT メールアドレス一致の購読情報を取得。
MailSubscriptionRepository.findTop200ByOrderByUpdatedAtDesc なし List<MailSubscription> SELECT 更新日時降順で最大200件取得。
MailSubscriptionRepository.findByCreatedAtBetween from:LocalDateTime, to:LocalDateTime List<MailSubscription> SELECT 作成日時が範囲内の購読情報を取得。
MailSubscriptionRepository.findByUnsubscribedAtBetween from:LocalDateTime, to:LocalDateTime List<MailSubscription> SELECT 解除日時が範囲内の購読情報を取得。

メソッド HTTP URL 説明
ReservationController.availability GET /api/reservations/availability コース・期間を指定して空き状況を取得
ReservationController.createReservation POST /api/reservations 予約を作成
MailSubscriptionController.register POST /api/mail/subscriptions/register メール購読を登録
MailSubscriptionController.unsubscribe POST /api/mail/subscriptions/unsubscribe メール購読を解除
AdminController.reservations GET /api/admin/reservations 管理用予約一覧を取得
AdminController.reservationSummary GET /api/admin/reservation-summary 管理用予約サマリを取得
AdminController.mailSubscriptions GET /api/admin/mail-subscriptions 管理用メール購読一覧を取得
AdminController.mailSummary GET /api/admin/mail-summary 管理用メール購読サマリを取得
AdminController.me GET /api/admin/me ログイン中管理ユーザー情報を取得
CsrfTokenController.csrfToken GET /api/csrf-token CSRFトークン情報を取得
RootRedirectController.root GET / ルートアクセスを /index.html へリダイレクト

+--------------------------- Controller Layer ---------------------------+
| [ReservationController]      [MailSubscriptionController]             |
| [AdminController]            [CsrfTokenController] [RootRedirectController] |
+-----------------------------------------------------------------------+
              |                        |
              | depends on             | (Csrf/RedirectはService非依存)
              v                        v
+---------------------------- Service Layer -----------------------------+
| [ReservationService]        [MailSubscriptionService]                 |
+-----------------------------------------------------------------------+
              |                              |
              | depends on                   | depends on
              v                              v
+--------------------------- Repository Layer ---------------------------+
| [ReservationRepository]  [SlotCapacityRepository]  [MailSubscriptionRepository] |
+-----------------------------------------------------------------------+
              |                    |                         |
              | handles            | handles                 | handles
              v                    v                         v
+----------------------------- Entity Layer -----------------------------+
| [Reservation]             [SlotCapacity]            [MailSubscription] |
+-----------------------------------------------------------------------+

Cross-cutting:
- Controller/Service は [DTOs] を介して入出力データを受け渡し
- [CsrfTokenController] は Spring Security の CsrfToken を直接利用
