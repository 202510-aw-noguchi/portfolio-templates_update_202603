作品概要

香水メーカーのWebサイトリニューアル。
申込数の低迷する香水メーカー主催のワークショップのてこ入れ策としてWebサイトのリニューアル依頼を受ける。

使用技術

フロントエンド：HTML / CSS / JavaScript

バックエンド：Java / Spring Boot

データベース：MySQL

その他：Git / Figma

担当範囲

企画提案・デザイン・フロントエンド・バックエンド

工夫した点・アピールポイント

既存顧客のUXを考慮し、2カラム構成を維持しながらマルチデバイス対応とすることで、スムーズな移行を狙う。ブランディングの観点から、同名ブランドとして実施するワークショップのトンマナを見直し（チープ→やや高級）。
